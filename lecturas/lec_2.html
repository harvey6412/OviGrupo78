Navegación interna y externa
(NAV -UL - OL - LI)

Para estos enlaces de navegación se usa la etiqueta < a> < /a> con un atributo href= "..." en el cual especifica la dirección de destino.

  • Navegación interna: Son aquellos enlaces de navegación que llevan a otra sección de la misma página y pueden definirse con anclas definidas con el atributo id o name.

  • Navegación Externa: Son los enlaces que dirigen hacia otras páginas que se encuentran fuera del sitio web actual.
    Por ejemplo:
    < a href=" https://www.google.com/"> Pagina Google< /a>

  • La etiqueta < nav>/nav> se define como el bloque principal de enlaces que necesita una página web para su navegación

  • Lista Desordenada
    UL: Esta la primera etiqueta con la que se debe abrir una lista y crear la lista desordenada
    LI: Esta etiqueta marca a las listas con viñetas, círculos negros, cuadrados de forma predeterminada

  • Lista Ordenada OL: Con esta etiqueta se debe abrir una lista y crear una lista ordenada LI: Al ser ordenada se listarán con números en forma consecutiva
