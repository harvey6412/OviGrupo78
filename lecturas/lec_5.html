Propiedades y selectores
css

    SELECTOR: define a los elementos HTML que se quieren realzar, estas van seguidas de una llave de apertura “{“ y otra de cierre “}”.

    Propiedad: es el tipo de estilo que se va a darle al elemento HTML, seguido de dos puntos (:). La propiedad debe ir dentro de las llaves {}, se puede agregar más de una propiedad en un selector eso depende de cómo se vaya mostrar el elemento de HTML.
    Las Propiedades básicas de HTML son:

    Propiedad fuente
    Propiedad Color
    Propiedad Fondo
    propiedad texto
    propiedad borde


  • Valor: se define como el estilo que se desea para el elemento HTML. Esta va seguida de un punto y coma.


  • La propiedad float para darle posicion a un texto o imagen es decir dejarlo in texto a la izquiera y una imagen a la derecha de una pagina.
    los siguientes valores son los mas usados:
    • left: el Contenido flota a la izquierda
    • right::el Contenido flota a la derecha
    • none:el contenido no flota se queda como esta.


  • La propiedad position es donde se define el tipo de posicion que se quiere dar a un elemento
    • static: el elemeto se queda inmovil
    • relative:: se adapta al espacio dependiendo el elemento
    • fixed: permanece en el mismo lugar asi si la pagina de desplaza y no deja espacios
    • absolute: su ubicaion es exacta sin afectar el contenido de la pagina.el contenido no flota se queda como esta.


  • La propiedad Overflow hace que el contenido cuando es muy grande se ajuste o en su defecto especifica si se debe hacer algún cambio como recortar o agregar barras que ayuden con el desplazamiento del mismo.
