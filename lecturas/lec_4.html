Estilos internos y externos
de CSS3

CSS por sus siglas en ingles Cascade Style Sheet u hojas de estilo en cascada, se basa en una serie de normas que rigen el estilo de los elementos en un documento, en otras palabras describe cómo se va a mostrar los elementos de un archivo HTML en pantalla, impresora o dispositivos basados en braille, se puede y aconseja la separación de los documentos escritos en HTML de la presentación del documento con las hojas de estilo incluyendo elementos como fondo, colores, márgenes, bordes, tipo de letra, etc.

  • Hoja de estilos externa: Es en la que se puede cambiar la vista de un sitio web completo cambiando un solo archivo, este debe ir con la extensión .css además cada página .html debe incluir la referencia de la hoja de estilo dentro de elemento < link> que se encuentra dentro de la etiqueta < head>:

  • < head>
    < link rel="stylesheet" type="text/css" href="style.css">
    < /head>
