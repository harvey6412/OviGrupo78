Introducción a html

Sus iniciales HTML por sus siglas en ingles HyperText Markup Language que significa Lenguaje de Marcado de Hipertexto, fue definido por la W3C (world wide web). Este es un lenguaje que es utilizado para la estructuración de páginas web, HTML no es un lenguaje de programación ya que no tiene funciones aritméticas, variables o estructuras propias de los lenguajes de programación como PHP. Los navegadores no muestran las etiquetas HTML , pero si las usa para representar el contenido de una pagina

Etiquetas de html

  • El elemento < !doctype html> Define que el documento esta bajo el estandar de HTML 5.
  • elemento < html> representa la raiz de un documento HTML. El resto de elementos descienden de este elemento.
  • etiqueta < head> Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo.
  • etiquetas < body> Representa el contenido principal de un documento HTML. Solo hay un elemento < body> en un documento.
  • etiqueta < title> Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página. Solamente puede contener texto y cualquier otra etiqueta contenida no será interpretada.
  • La etiqueta < link> Usada para enlazar JavaScript y CSS externos con el documento HTML actual.
  • La etiqueta < meta> Define los metadatos que no pueden ser definidos usando otro elemento HTML.
  • La etiqueta style Etiqueta de estilo usada para escribir CSS en línea.
