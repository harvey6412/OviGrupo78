OBJETIVOS DEL OVI

OBJETIVO GENERAL
Exponer teóricamente las herramientas de HTML y CSS para que sean aplicados de forma correcta en el desarrollo de sitios web.

OBJETIVOS ESPECIFICOS
1. Mostrar al lector la estructura básica de HTML y CSS.
2. Definir el contenido de los estándares HTML y posicionamiento de elementos CSS en las páginas web.
3. Enseñar la documentación para que el estudiante tenga buenas prácticas de programación al momento del desarrollo.
