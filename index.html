Introducción OVI

El proposito de este proyecto es crear una pagina web desde cero y de manera satisfactoria desarrollar todos los items propuestos por el docente en la guia de actividades; siendo el proceso una manera didactica de aprendizaje en que el estudiante basandonse en una maquetación ejecutara el desarrollo de dicha pagina, pondrá en practica los conocimientos hmtl y css, ademas de esto contará con ayudas para que se cumpla con la Tarea.

Este objeto virtual de información se creo para que el estudiante con ayuda de sus compañeros visualice situaciones a las que se podria enfrentar en un entorno laboral, enfocandose en las necesidades e intereses de sus futuros proyectos laborales, esto le ayudara a tener una mejor perpectiva del mundo real.

La pagina debe contar con información sobre el desarrollo de sitios web, ya que este es tema es muy grande el estudiante debe eligir los temas que va tratar y dar una lectura o descripcion de las mismas dada su importancia.

Objetivo General

Crear una pagina web de forma colaborativa donde se expongan contenido teorico de las herramientas de HTML y CSS para que sean aplicados de forma correcta en el desarrollo de sitios web.

Objetivos Específicos

  • Desarrolar un sitio web donde los integrantes del grupo apoyaran a la contrucción del sitio web para que este sea visualizado de manera correcta.

  • Mostrar al usuario la estructura basica de html y css. por medio de texto donde se describa de manera adecuada los items elegidos.

  • Exponer por medio de contenido multimendia (videos- imagenes) contenido anexo a los temas que se van tratar.

  • Ayudar a que el usuario con actividades interactivas ponga en practica y refuerce los conocimientos adquiridos en esta pagina web.
