Introducción OVI

El propósito de este proyecto es crear una página web desde cero y de manera satisfactoria desarrollar todos los ítems propuestos por el docente en la guía de actividades; siendo el proceso una manera didáctica de aprendizaje en que el estudiante basándose en una maquetación ejecutará el desarrollo de dicha página, pondrá en práctica los conocimientos HTML y CSS, además de esto contará con ayudas para que se cumpla con la Tarea.

Este objeto virtual de información se creó para que el estudiante con ayuda de sus compañeros visualice situaciones a las que se podría enfrentar en un entorno laboral, enfocándose en las necesidades e intereses de sus futuros proyectos laborales, esto le ayudara a tener una mejor perspectiva del mundo real.

La página debe contar con información sobre el desarrollo de sitios web, ya que este es tema es muy grande el estudiante debe elegir los temas que va tratar y dar una lectura o descripción de las mismas dada su importancia.

Objetivo General

Crear una página web de forma colaborativa donde se expongan contenido teórico de las herramientas de HTML y CSS para que sean aplicados de forma correcta en el desarrollo de sitios web.

Objetivos Específicos

  • Desarrollar un sitio web donde los integrantes del grupo apoyaran a la construcción del sitio web para que este sea visualizado de manera correcta.

  • Mostrar al usuario la estructura básica de HTML y CSS. por medio de texto donde se describa de manera adecuada los ítems elegidos.

  • Exponer por medio de contenido multimedia (videos- imágenes) contenido anexo a los temas que se van tratar.

  • Ayudar a que el usuario con actividades interactivas ponga en práctica y refuerce los conocimientos adquiridos en esta página web.
